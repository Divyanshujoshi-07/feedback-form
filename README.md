# 🎓 Student Feedback Form

## 📌 Project Overview

The **Student Feedback Form** is a modern, responsive, and interactive web application developed using **HTML, CSS, and JavaScript**. The project is designed to collect valuable feedback from students regarding their academic experience, teaching quality, campus facilities, and overall satisfaction.

The application features a multi-step form interface that makes the feedback process simple, organized, and user-friendly. Interactive components such as star ratings, mood selection, sliders, and text feedback fields enhance user engagement and provide a better user experience.

---

## ✨ Features

### 📝 Multi-Step Feedback Form
- Organized into multiple sections for better usability.
- Progress indicator to track completion status.
- Reduces form complexity and improves user experience.

### ⭐ Interactive Rating System
- Star-based rating system for evaluating services and faculty performance.
- Easy and intuitive to use.

### 😊 Mood Selection
- Allows students to express their overall experience through mood icons.

### 📊 Facility & Wi-Fi Rating Slider
- Interactive slider for rating campus facilities and internet services.

### 🔤 Character Counter
- Displays the number of characters entered in feedback text areas.
- Encourages concise and meaningful feedback.

### ✅ Form Validation
- Ensures all required fields are completed.
- Prevents invalid or incomplete submissions.

### 🎨 Modern UI Design
- Attractive layout with animations and smooth transitions.
- Professional and visually appealing interface.

### 📱 Responsive Design
- Compatible with desktops, tablets, and mobile devices.
- Automatically adjusts to different screen sizes.

## 🚀 How It Works

1. Open the Student Feedback Form.
2. Enter the required student information.
3. Rate faculty, courses, and campus facilities.
4. Select a mood that reflects your overall experience.
5. Provide suggestions and additional comments.
6. Submit the form after completing all sections.
7. Form validation ensures data accuracy before submission.
